# Editors & Windows Run Commands Cheat Sheet

## üìù Vim Editor Cheat Sheet

### Modes
```
Normal Mode    - Default, for navigation (press ESC)
Insert Mode    - For typing text (press i, a, o)
Visual Mode    - For selecting text (press v, V, Ctrl+v)
Command Mode   - For commands (press :)
```

### Basic Operations
```vim
# Opening/Saving/Quitting
vim file.txt           # Open file
:w                     # Save (write)
:w newfile.txt         # Save as
:q                     # Quit
:q!                    # Quit without saving
:wq  or  :x            # Save and quit
ZZ                     # Save and quit (Normal mode)
:e file.txt            # Open another file
:e!                    # Reload current file
```

### Navigation (Normal Mode)
```vim
# Basic Movement
h                      # Left
j                      # Down
k                      # Up
l                      # Right

# Word Movement
w                      # Next word start
b                      # Previous word start
e                      # Next word end
W, B, E                # Same, but skip punctuation

# Line Movement
0                      # Start of line
^                      # First non-blank character
$                      # End of line
g_                     # Last non-blank character

# Screen Movement
H                      # Top of screen
M                      # Middle of screen
L                      # Bottom of screen
Ctrl+u                 # Half page up
Ctrl+d                 # Half page down
Ctrl+b                 # Full page up
Ctrl+f                 # Full page down

# File Movement
gg                     # First line
G                      # Last line
5G  or  :5             # Go to line 5
%                      # Jump to matching bracket
```

### Editing (Insert Mode)
```vim
# Enter Insert Mode
i                      # Insert before cursor
a                      # Insert after cursor
I                      # Insert at line start
A                      # Insert at line end
o                      # Open new line below

s                      # Delete char and insert
S                      # Delete line and insert
ESC                    # Exit insert mode

# In Insert Mode
Ctrl+h                 # Delete previous char
Ctrl+w                 # Delete previous word
Ctrl+u                 # Delete to line start
```

### Visual Mode
```vim

# In Visual Mode

```

### Search and Replace
```vim
# Search
/pattern               # Search forward
?pattern               # Search backward
n                      # Next match
N                      # Previous match
*                      # Search word under cursor (forward)
#                      # Search word under cursor (backward)
:noh                   # Clear search highlighting

# Replace
:s/old/new             # Replace first on line
:s/old/new/g           # Replace all on line
:%s/old/new/g          # Replace all in file
:%s/old/new/gc         # Replace all with confirmation
:5,10s/old/new/g       # Replace in lines 5-10
```

### Multi-File Operations
```vim
:e file2.txt           # Open file
:ls  or  :buffers      # List buffers
:bn                    # Next buffer
:bp                    # Previous buffer
:b3                    # Go to buffer 3
:bd                    # Delete buffer (close file)

# Split Windows
:split  or  :sp        # Horizontal split
:vsplit  or  :vsp      # Vertical split
Ctrl+w h/j/k/l         # Navigate splits
Ctrl+w w               # Cycle through splits
Ctrl+w q               # Close split
Ctrl+w =               # Equal size splits
Ctrl+w _               # Maximize height
Ctrl+w |               # Maximize width

# Tabs
:tabnew file.txt       # Open in new tab
:tabn                  # Next tab
:tabp                  # Previous tab
:tabc                  # Close tab
gt                     # Next tab
gT                     # Previous tab
```

### .vimrc Configuration
```vim
" ~/.vimrc - Vim configuration file

" General Settings
set nocompatible              " Disable vi compatibility
set number                    " Show line numbers
set relativenumber            " Relative line numbers
set ruler                     " Show cursor position
set showcmd                   " Show command in status line
set wildmenu                  " Command-line completion
set scrolloff=8               " Keep 8 lines visible above/below cursor

" Indentation
set autoindent                " Auto indent
set smartindent               " Smart indent
set tabstop=4                 " Tab width
set shiftwidth=4              " Indent width
set expandtab                 " Use spaces instead of tabs
set smarttab                  " Smart tab

" Search
set hlsearch                  " Highlight searches
set incsearch                 " Incremental search
set ignorecase                " Case-insensitive search
set smartcase                 " Smart case search

" Display
syntax on                     " Syntax highlighting
set background=dark           " Dark background
colorscheme desert            " Color scheme
set cursorline               " Highlight current line
set showmatch                " Highlight matching brackets

" Performance
set lazyredraw               " Don't redraw during macros
set ttyfast                  " Fast terminal connection

" Key Mappings
" Leader key
let mapleader = ","

" Quick save
nnoremap <leader>w :w<CR>

" Quick quit
nnoremap <leader>q :q<CR>

" Clear search highlighting
nnoremap <leader>h :noh<CR>

" Easy window navigation
nnoremap <C-h> <C-w>h
nnoremap <C-j> <C-w>j
nnoremap <C-k> <C-w>k
nnoremap <C-l> <C-w>l

" Move lines up/down
nnoremap <A-j> :m .+1<CR>==
nnoremap <A-k> :m .-2<CR>==
vnoremap <A-j> :m '>+1<CR>gv=gv
vnoremap <A-k> :m '<-2<CR>gv=gv

" Plugins (using vim-plug)
" Install vim-plug first:
" curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
"     https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

call plug#begin('~/.vim/plugged')
Plug 'preservim/nerdtree'         " File explorer
Plug 'tpope/vim-fugitive'          " Git integration
Plug 'airblade/vim-gitgutter'      " Git diff in gutter
Plug 'vim-airline/vim-airline'     " Status line
Plug 'junegunn/fzf.vim'            " Fuzzy finder
call plug#end()

" NERDTree settings
nnoremap <leader>n :NERDTreeToggle<CR>
let NERDTreeShowHidden=1
```

## üìù Nano Editor Cheat Sheet

### Basic Operations
```bash
# Opening Files
nano file.txt          # Open file
nano +10 file.txt      # Open at line 10
nano -w file.txt       # No auto line wrap
nano -B file.txt       # Backup before editing
```

### Key Commands (^ means Ctrl)
```
# File Operations
^O                     # Write Out (Save)
  Enter                #   Confirm filename
^X                     # Exit
^R                     # Read file (insert file)
^T                     # To Files (file browser)

# Navigation
^A                     # Beginning of line
^E                     # End of line
^Y                     # Page Up
^V                     # Page Down
^_                     # Go to line/column number
^C                     # Show cursor position

# Editing
^K                     # Cut line
^U                     # Paste (Uncut)
^J                     # Justify paragraph
^6                     # Mark text (start selection)
M-6                    # Copy marked text
Alt+A                  # Mark text (alternative)
^]                     # Go to matching bracket

# Search and Replace
^W                     # Where Is (Search)
  Enter                #   Find next
Alt+W                  # Find Next
^\\                    # Replace
  Enter                #   Confirm old text
  Enter                #   Enter new text
  Y/N/A                #   Yes/No/All

# Delete
^K                     # Delete line
Backspace              # Delete character before
Delete                 # Delete character under
Alt+Del                # Delete word to the left
^Del                   # Delete word to the right

# Undo/Redo
Alt+U                  # Undo
Alt+E                  # Redo

# Multiple Buffers
Alt+<                  # Previous buffer
Alt+>                  # Next buffer

# Spell Check
^T                     # Spell checker (if available)

# Help
^G                     # Help menu
```

### Command Line Options

```bash
nano [options] file.txt

-Q "string"            # Quote string
-R                     # Restricted mode
-S                     # Smooth scrolling
-T number              # Tab size
-U                     # Quick blanking
-V                     # Print version
-W                     # Detect word boundaries
-Y name                # Syntax highlighting
-c                     # Constantly show cursor position
-h                     # Help
-i                     # Auto indent
-k                     # Cut to end
-l                     # Don't follow symlinks
-m                     # Mouse support
-n                     # No conversion from DOS/Mac
-o directory           # Operating directory
-p                     # Preserve XON/XOFF
-r number              # Hard-wrap at column number
-s "program"           # Spell checker program

```

### .nanorc Configuration
```bash
# ~/.nanorc - Nano configuration file

# Settings

set backup             # Create backup files
set backupdir ~/.nano/backups  # Backup directory
set casesensitive      # Case-sensitive searches
set cut                # Cut to end with ^K
set historylog         # Save search/replace history
set linenumbers        # Show line numbers
set mouse              # Enable mouse support
set multibuffer        # Allow multiple files
set nohelp             # Don't show help bar
set nowrap             # Don't wrap long lines
set smooth             # Smooth scrolling
set suspend            # Allow Ctrl+Z to suspend
set tabsize 4          # Tab width
set tabstospaces       # Convert tabs to spaces

# Syntax Highlighting
include /usr/share/nano/*.nanorc
include /usr/share/nano/extra/*.nanorc

# Key bindings
bind ^S savefile main
bind ^Q exit all
bind ^F whereis all
bind ^H help all
```

## üîß Troubleshooting

### Vim Issues
```vim
" Vim not responding
Ctrl+Q                 " Try instead of Ctrl+S (XON/XOFF)

" Undo tree corrupted
:earlier 1f            " Go to state in previous file write

" Wrong file encoding
:e ++enc=utf8          " Reload with UTF-8

" Swap file error
:e                     " Edit again
:recover               " Or recover from swap
rm .file.swp           " Or delete swap file
```

### Nano Issues
```bash
# Nano not found
apt-get install nano   # Debian/Ubuntu
yum install nano       # RHEL/CentOS

# Config not loading
nano -I                # Ignore nanorc files
```

### VS Code Issues
```bash
# Reset settings
File > Preferences > Settings
  Click "Open Settings (JSON)"
  Delete content and save

# Clear cache
# Windows: %APPDATA%\Code\Cache
# Linux: ~/.config/Code/Cache
rm -rf ~/.config/Code/Cache/*

# Reinstall extensions
code --list-extensions
code --uninstall-extension extension.id
code --install-extension extension.id

# Disable GPU acceleration (if crashing)
code --disable-gpu
```

## üéì Interview Questions

**Q: When would you use Vim over other editors?**
- Remote server administration (SSH)
- Limited resources (low RAM/CPU)
- Need for efficiency in text manipulation
- Terminal-only environments
- Configuration file editing

**Q: What are Vim modes?**
- Normal: Navigation and commands (default)
- Insert: Typing text (i, a, o)
- Visual: Selecting text (v, V, Ctrl+v)
- Command: Ex commands (:w, :q)

**Q: How to exit Vim?**
- :q (quit)
- :q! (quit without saving)
- :wq or :x (save and quit)
- ZZ (save and quit, normal mode)

**Q: VS Code vs IDE (like Visual Studio)?**
- VS Code: Lightweight, extensible, cross-platform
- IDE: Feature-rich, language-specific, heavier
- VS Code better for web dev, scripting
- IDE better for large enterprise applications

**Q: How to search and replace in Vim?**
- :s/old/new/g (current line)
- :%s/old/new/g (entire file)
- :%s/old/new/gc (with confirmation)
- :5,10s/old/new/g (lines 5-10)

**Q: What is .vimrc?**
- Vim configuration file
- Located in home directory (~/.vimrc)
- Contains settings and key mappings
- Executes on Vim startup

**Q: VS Code remote development?**
- Remote-SSH extension
- Edit files on remote servers
- Full IntelliSense and debugging
- Port forwarding support

**Q: Nano vs Vi/Vim for beginners?**
- Nano: Easier, visible commands, no modes
- Vim: More powerful but steeper learning curve
- Nano for quick edits, Vim for productivity

**Q: Common Windows Run commands?**
- ncpa.cpl: Network connections
- services.msc: Services management
- eventvwr.msc: Event viewer
- devmgmt.msc: Device manager
- mstsc: Remote desktop

**Q: How to edit files as sudo in Vim?**
```vim
:w !sudo tee %         # Save without reopening
sudo vim file.txt      # Or open with sudo initially
```

**Q: VS Code debugging advantages?**
- Visual breakpoints
- Variable inspection
- Call stack navigation
- Integrated terminal
- Multi-language support

**Q: Best practices for server config editing?**
1. Backup before editing: cp config config.bak
2. Test syntax: nginx -t, apache2ctl configtest
3. Use version control for configs
4. Document changes in comments
5. Test changes before committing
