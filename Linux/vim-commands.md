# Essential Vim Commands

Basic navigation and editing commands in Vim.

## Navigation
- `h, j, k, l` - left, down, up, right
- `w` - next word
- `b` - previous word
- `0` - beginning of line
- `$` - end of line

## Editing
- `i` - insert mode
- `a` - append mode
- `o` - new line below
- `dd` - delete line
